# BuyLottoX Frontend

React + Vite frontend for the BuyLottoX Next.js backend.

## Features
- Login / Register
- USER dashboard
- Buy ticket
- My tickets
- Create payment
- Upload payment slip
- ADMIN dashboard
- Users list
- User history
- Payment review and approval
- Draw management

## Run
```bash
npm install
copy .env.example .env
npm run dev
```

The Vite dev server proxies `/api` to `http://localhost:4000` by default.
