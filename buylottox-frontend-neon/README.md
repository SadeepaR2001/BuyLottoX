# BuyLottoX Frontend (React + Vite + TS)

## Setup
```bash
npm install
npm run dev
```

## Notes
- This project is **frontend-only** for now. It uses **mock services** under `src/mock/`.
- Later, you can switch to real NestJS APIs by updating `src/services/data/index.ts` and enabling Axios in `src/services/http/client.ts`.

## Default rules (change anytime)
- Ticket numbers: **6 numbers from 1–49**
